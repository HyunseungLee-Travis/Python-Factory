# mut 변수

`mut`란 변수의 값을 바꿀 수 있다는 뜻입니다.

immutable 변수는 값을 바꿀 수 없는데 Rust는 기본적으로 immutable입니다.

그리하여 변수에 값을 저장하고 나중에 바꾸고 싶다면 `mut`를 사용해야 합니다.

```rust
let mut mood = "happy";
```

## mut 사용 방법

식별자 (예: `let`) 다음에 `mut`를 붙이면 mutable 변수로 인식합니다.

값을 지정한 후 다시 값을 바꿀 수 있습니다.
