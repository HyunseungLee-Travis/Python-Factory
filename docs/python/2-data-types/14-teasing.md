# 친구 놀리기 프로젝트

친구를 놀리는 방법을 알아보겠습니다.

친구의 이름을 입력하면 ` 바보` 가 이름 다음에 출력되게 만들겠습니다.

## 코드

`name = input()` 코드를 실행하면 `name`이라는 변수 안에 입력된 친구의 이름이 저장됩니다.

`print(name, "바보")`를 실행하면 친구의 이름과 `바보`를 출력합니다.
