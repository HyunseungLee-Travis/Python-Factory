# 정렬

리스트는 정보를 저장하는 데이터 데이터 형태의 종류입니다.

정렬도 정보를 저장하는 데이터 형태의 종류입니다.

정렬은 리스트와 비슷하지만 조금 다른 점들이 있습니다.

1. 정렬은 파이썬에서 자체적으로 사용할 수 없어서 `numpy` 모듈을 사용해야 합니다.
2. 정렬은 아주 큰 데이터를 효율적으로 저장할 수 있습니다.
3. 정렬은 숫자를 저장하는 데이터 종류로 더 편합니다.
