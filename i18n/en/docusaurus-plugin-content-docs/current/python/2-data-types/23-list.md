# 리스트

리스트는 한국어로 '순서'입니다.

파이썬에서 리스트는 다양한 데이터를 순서대로 나열한 데이터 형태를 뜻합니다.

예를 들면, 일상생활에서 오늘 해야할 일을 나열할 때 파이썬에서 리스트를 사용할 수 있습니다.
